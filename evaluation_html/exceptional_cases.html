My demo app won't start!

If you have trouble running the applications you've downloaded for the WarmUp Task or Task 1, first try to force stop the previous run, if any, as instructed in the previous tasks: "After you exit or close the APP, the process running them might get stuck. You will need to identify it in your operating system's task manager and force close it. Depending on your OS, it will be either the tasks name (WarmUpTask, ALFTask or MyGame) or simply an instance of your local JRE run."

If that did not work and you are getting errors when trying to run the APP, follow the next instructions!

To get started, you will need to get your hands on 2 gems:

  • 1. The JMonkeyEngine development platform, a NetBeans based IDE.
  • 2. The EgoSim framework from my git repository.

Unzip the JMonkleyEngine Platform and run it. On the first run you will be asked to configure the working directory. Please do so!

From the File menu, select the "Import Project > From ZIP..." option. Point to the location of the EgoSim zip you've recently downloaded. This will import the BodyCentricSim project in your workspace.

To run the WarmUp Task, from the "Source Packages", expand the "dk.itu.bodysim" package and open the "WarmUpTask.java" class. Run it by hitting the SHIFT+F6 key combination.

To run Task 1, from the "Source Packages", expand the "dk.itu.bodysim" package and open the "ALFTask.java" class. Run it by hitting the SHIFT+F6 key combination.

NB: After you exit or close the APP, the process running it might get stuck. Identify the following icon in the JMonkeyEngine IDE and press the force close button.
